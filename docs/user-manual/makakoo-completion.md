# `makakoo completion`

Generate and install shell completion for bash / zsh / fish / elvish / powershell. Auto-detects your shell when run with no args.

## Example

```sh
makakoo completion install        # auto-detect
makakoo completion zsh >> ~/.zshrc
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo completion --help
```
