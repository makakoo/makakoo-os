# `makakoo distro`

List, install, or switch Makakoo distros. A distro is a named bundle of plugins. Shipped distros: `minimal`, `core` (default), `sebastian`, `creator`, `trader`.

## Example

```sh
makakoo distro list
makakoo distro install creator
makakoo distro show core
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo distro --help
```
