# `makakoo mcp`

Manage the Makakoo MCP stdio server (`makakoo-mcp`) — the protocol bridge that exposes Makakoo's tools to any MCP client (Claude Desktop, Claude Code, etc.).

## Example

```sh
makakoo mcp install
makakoo mcp uninstall
makakoo mcp status
makakoo mcp --help
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo mcp --help
```
