# `makakoo plugin`

Install, list, update, or remove plugins from the Makakoo plugin ecosystem. Sources: local path, git URL (git+https://…), core-shipped bundle.

## Example

```sh
makakoo plugin list
makakoo plugin install git+https://github.com/you/my-plugin@v1
makakoo plugin remove my-plugin
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo plugin --help
```
