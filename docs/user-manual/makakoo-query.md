# `makakoo query`

Ask the Brain a question. Full-text + vector search, then LLM synthesis of a short answer referencing the hit pages.

## Example

```sh
makakoo query "what did I decide about the database migration?"
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo query --help
```
