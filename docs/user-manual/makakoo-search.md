# `makakoo search`

Full-text search over the Brain. FTS5-backed, returns raw hits with snippet context. Faster than `query` (no LLM call) and lets you grep for exact phrases.

## Example

```sh
makakoo search "polymarket"
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo search --help
```
