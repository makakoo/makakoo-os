# `makakoo status`

One-line-per-subsystem health summary. Good as a post-install sanity check or when something feels broken. Deeper: `makakoo health --verbose`.

## Example

```sh
makakoo status
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo status --help
```
