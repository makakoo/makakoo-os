# `makakoo uninfect`

Remove the Makakoo bootstrap block from CLI host config files. Symmetric to `infect`. Leaves user prose above and below the block intact.

## Example

```sh
makakoo uninfect --dry-run
makakoo uninfect
```

## Full command surface

Run with `--help` for every flag, subcommand, and exit code:

```sh
makakoo uninfect --help
```
