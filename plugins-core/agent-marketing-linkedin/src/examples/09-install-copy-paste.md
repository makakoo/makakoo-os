The install is one line you paste into any AI agent.

Read https://raw.githubusercontent.com/traylinx/lope/main/INSTALL.md and follow the instructions to install lope on this machine natively.

That's it. The agent fetches INSTALL.md, runs the six steps, auto-detects Claude Code, Codex, Gemini CLI, OpenCode, and Cursor on your machine, and wires the slash commands into each host's native directory in the format that host expects. About 30 seconds end to end. Reports back when it's live.

No package manager. No broken wheels. No Python venv dance. No "please install these 47 transitive dependencies." Pure Python stdlib, zero external deps.

If you've been burned by pip install turning into an hour of debugging somebody else's CUDA version, this is the explicit rebellion. One paste, one agent, done.
