## New rebuild-class issues in v2.0

**N1 — D20 session handoff hand-wave.** INSTALL_MATRIX.md has no
session/transcript column. Each host stores transcripts differently.
Either add ~1 week per-host transcript parsing to Phase F, OR demote
D20 to v0.2, OR reshape v0.1 scope to manual `harvey remember` with
passive watching deferred.

**N2 — D10 macOS path contradiction.** D10 text says `default_home()`
returns `~/MAKAKOO` on macOS. Section 5 + section 10 show runtime at
`~/.makakoo/`. Pick one.

**N3 — Event bus still missing.** EventBus in §5 directory listing but
no D-number, no semantics, no delivery guarantees. Add D21 or remove
from core.
